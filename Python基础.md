## 变量和简单的数据类型

- ### 变量

  - ```python
    # 用变量存储字符串,再打印这个变量.
    message = "Hello Python world!"
    print(message)
    ```

  - ```python
    message = "Hello Python world!"
    print(message)
    message = "Hello Python Crash Course world!"
    print(message)
    # Hello Python world!
    # Hello Python Crash Course world!
    # 变量赋值会覆盖之前的.在程序中可随时修改变量的值，而Python将始终记录变量的最新值
    ```
